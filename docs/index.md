# pypi-example

## Install

```python
pip install cubao-pypi-example -U
```

## Usage

```python
python3 -m cubao_pypi_example
python3 -m cubao_pypi_example hello_world
python3 -m cubao_pypi_example list_files
python3 -m cubao_pypi_example list_files /tmp
```
